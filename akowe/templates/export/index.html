Export for Tax Preparation

Export your financial data in formats compatible with popular Canadian tax preparation software.

CRA T2125 Format

Export in a format optimized for completing the CRA T2125 form (Statement of Business Activities).

TurboTax Format

Export in a format optimized for importing into TurboTax.

Best compatible with TurboTax Desktop versions. For TurboTax Online, manual data entry may be required.
Wealthsimple Tax Format

Export in a format optimized for importing into Wealthsimple Tax (formerly SimpleTax).

T2 GIFI Format

Export in GIFI (General Index of Financial Information) format for corporate T2 tax returns.

T2 Schedule 8 Format

Export in Capital Cost Allowance (CCA) Schedule 8 format for T2 corporate tax returns.

Corporate TurboTax Format

Export in a format optimized for importing into TurboTax Business/Corporate edition.

For use with TurboTax Business or other T2 corporate tax software.
About Export Data

The exported CSV files can be used for:

  • Importing into spreadsheet applications (Excel, Google Sheets, etc.)
  • Sharing with your accountant or tax preparer
  • Creating custom reports or analyses
  • Backing up your financial data
  • Direct import into Canadian tax preparation software (new!)
All dates are in ISO 8601 format (YYYY-MM-DD) and monetary amounts include two decimal places.

Import Expenses

About Import Data
Important: Importing data will add new records to your database. It will not update or replace existing records.

Use the import feature for:

  • Restoring data from a previous export
  • Migrating data from another system
  • Bulk adding records prepared in a spreadsheet

For the "All Transactions" import, use the file created with the "Export All Transactions" feature. This will correctly handle both income and expense records.
